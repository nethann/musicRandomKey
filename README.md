# Random Music Key Generator

A simple Electron desktop app that generates a "daily music key" (Major or Minor) and provides a quick YouTube link for backing tracks.

---

## Features

- Generates a random music key (A through G, including sharps) and whether itâ€™s Major or Minor.
- Provides a clickable YouTube link to backing tracks for the generated key.
- Daily key resets every 24 hours.
- Simple, minimal interface with a big circular "Generate" button.
